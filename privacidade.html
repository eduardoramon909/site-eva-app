Política de Privacidade

Última atualização: 02 de Novembro de 2025

A Ed Company ("nós", "nosso") opera o aplicativo móvel Eva Assistente Financeira ("Aplicativo"). Esta página informa sobre nossas políticas relativas à coleta, uso e divulgação de dados pessoais quando você usa nosso Aplicativo.

Sua privacidade é fundamental para nós. Não compartilhamos ou vendemos suas informações pessoais.

1. Coleta e Uso de Dados

Coletamos diferentes tipos de informações para fornecer e melhorar nosso Aplicativo para você.

Tipos de Dados Coletados

  • Dados Pessoais e Financeiros Fornecidos por Você: Ao usar o Eva Assistente Financeira, pedimos que você forneça dados financeiros como receitas, despesas, metas e categorias. Esses dados são a base do funcionamento do aplicativo.
  • Dados de Autenticação (Firebase): Usamos o Firebase Authentication (Google) para criar e gerenciar sua conta de forma segura. Isso pode incluir seu e-mail ou método de login social (Google, etc.). Não temos acesso direto à sua senha.
  • Dados de Pagamento (Stripe): Para a assinatura "Eva Premium", utilizamos o Stripe como nosso processador de pagamentos. Não coletamos, armazenamos ou temos acesso aos dados do seu cartão de crédito. Essas informações são enviadas diretamente ao Stripe.

2. Armazenamento de Dados (Firebase Firestore)

Para permitir a sincronização entre dispositivos e fornecer backup, seus dados financeiros (transações, metas, etc.) são armazenados de forma segura nos servidores do Firebase Firestore (um serviço do Google). Utilizamos as regras de segurança do Firestore para garantir que apenas você tenha acesso de leitura e escrita aos seus próprios dados.

3. Uso dos Dados

Usamos os dados coletados para:

  • Fornecer e manter o Aplicativo;
  • Gerenciar sua conta e sua assinatura Premium;
  • Sincronizar seus dados entre seus dispositivos;
  • Fornecer suporte ao cliente.

4. Compartilhamento de Dados

Não compartilhamos seus dados financeiros pessoais com terceiros. As únicas entidades que têm acesso a *partes* dos seus dados são os provedores de serviço essenciais para o funcionamento do app:

  • Google (Firebase): Para autenticação e armazenamento de dados na nuvem.
  • Stripe: Para processamento seguro de pagamentos da assinatura Premium.

5. Segurança

A segurança dos seus dados é nossa prioridade. Confiamos nas medidas de segurança robustas do Firebase e do Stripe para proteger suas informações. No entanto, lembre-se de que nenhum método de transmissão pela Internet ou armazenamento eletrônico é 100% seguro.
